## 📋 **INSTALLATION STEPS**

### **1. Profile Repository**

```bash
# Go to your profile repo
https://github.com/rajendrabist07/rajendrabist07

# Click on README.md
# Click edit (pencil icon)
# Delete everything
# Paste the code above
# Commit changes
```

### **2. Enable Snake Animation** (Optional but recommended)

```bash
# Create GitHub Action for snake
# Go to your profile repo
# Create folder: .github/workflows
# Create file: snake.yml
```

**snake.yml**

```yaml
name: Generate snake animation

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: Platane/snk@v2
        id: snake-gif
        with:
          github_user_name: rajendrabist07
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v2.1.3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
